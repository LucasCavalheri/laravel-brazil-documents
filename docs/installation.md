# Instalação

## Requisitos

- PHP 8.3+
- Laravel 13.x

## Composer

```bash
composer require cavalheri/laravel-brazil-documents
```

O pacote é descoberto automaticamente pelo Laravel. Não é necessário registrar manualmente.

## Publicar arquivos (opcional)

```bash
php artisan vendor:publish --tag=brazil-documents-config
php artisan vendor:publish --tag=brazil-documents-lang
```
