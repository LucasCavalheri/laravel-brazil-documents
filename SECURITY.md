# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **lucas.dev.carvalho@gmail.com** with:

- A description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Suggested fix (optional)

You can expect an initial response within **7 days**. If the issue is accepted, a fix will be coordinated and released on Packagist with a changelog entry.

Thank you for helping keep **laravel-brazil-documents** safe for everyone.
